# segmentum

Segmentum is Segment.com open-source alternative. **WORK IN PROGRESS**

## Linting
`lein kibit`<br/> <br/>
`lein eastwood`

## Prerequisites

You will need [Leiningen][1] 2.0 or above installed.

[1]: https://github.com/technomancy/leiningen

## Running

To start a web server for the application, run:

    lein run
